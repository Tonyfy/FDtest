# 人脸检测

seetaface是由中科院计算所推出的人脸算法引擎。本项目意在使用人脸检测的部分，并评估性能等。

## 配置与使用
 
* 克隆本项目到本地
* 使用vs2013打开FDtest.sln
* 配置好合适的 opencv 库和 vs 项目
* 重新编译使用，也可使用cmake重新编译
